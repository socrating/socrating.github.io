在linux上运行clash

Author: John Doe Date: March 5, 2022  14:20:46

由于 Linux 发行版版本众多，我们无法提供所有发行版的安装说明，下文以 Ubuntu Desktop 20.04 LTS为例。（作为一个 Linux 用户，您应该明白如何安装。）

在终端执行 cd && mkdir clash ，创建 clash 文件夹。

在终端执行 cd && cd clash ，进入 clash 文件夹。

下载 Clash 文件，撰写此教程时最新版本为v1.41，所以下载命令为 wget -N –no-check-certificate https://github.com/Dreamacro/clash/releases/download/v1.4.1/clash-linux-amd64-v1.4.1.gz 。

在终端执行 gunzip clash-linux-amd64-v1.4.1.gz ，解压.gz文件。

将解压后的文件重命名为 clash。

在终端执行 chmod +x clash 赋予 Clash 执行权限

复制托管链接

在终端执行 wget -O config.yaml 刚刚复制的托管链接

在终端执行 ./clash -d . 即可启动 Clash，同時启动 HTTP 代理和 Socks5 代理。

访问 Clash Dashboard 可以进行切换节点、延迟检测等操作。Host和端口请按照下图设置。

设置代理

打开网络代理设置-有线设置，网络代理-选择手动，填写 HTTP 和 HTTPS 代理为 127.0.0.1:7890 ，填写Socks主机为 127.0.0.1:7891 ，即可启用系统代理。
